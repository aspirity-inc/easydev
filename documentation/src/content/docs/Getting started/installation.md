title: Installation

## Default installation
Run one of the following commands to add EasyDev to your project:

```bash
npm install easydev
```

or

```bash
yarn add easydev
```

## With styled-components

Run one of the following commands to add EasyDev to your project:

```bash
npm install easydev styled-components
```

or

```bash
yarn add easydev styled-components
```
