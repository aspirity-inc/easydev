---
title: Main
---

The Main is a part of Layout component. Here you can add your main content.

```tsx
<Main
  style={{
    minHeight: 300,
  }}
>
  <Center>Main</Center>
</Main>
```

## CSS

The following class names are useful for styling with CSS.

| Class         | Description                         |
| ------------- | ----------------------------------- |
| **easy_main** | Styles applied to the root element. |
